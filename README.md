# Multi-Agent System Web Platform (Showcase Version)

Веб-платформа для создания, настройки и выполнения сценариев взаимодействия интеллектуальных ИИ-агентов с ролевыми моделями.

> [!IMPORTANT]
> **Информация для проверяющего (Showcase-дисклеймер):**
> Этот репозиторий подготовлен специально для демонстрации работодателю в рамках собеседования.
> * **Цель репозитория:** Показать стиль кодирования (Code Style), культуру документирования, проектирование асинхронных потоков данных (FastAPI SSE, LangGraph Streaming) и архитектурные решения.
> * **Полнота кода:** Репозиторий содержит ключевые модули бэкенда демонстрационной архитектуры (логику агентов, песочницу выполнения кода и движок RAG). Полная версия платформы является коммерческой закрытой системой и содержит клиентскую часть (Next.js), схемы данных, файлы аутентификации и миграции базы данных, которые здесь опущены.

---

## 📁 Структура демонстрационных файлов

В этом репозитории представлены следующие ключевые компоненты бэкенд-системы:

* **[app/agents.py](app/agents.py)** — Реализация асинхронных эндпоинтов для взаимодействия с агентами и стриминга логов выполнения LangGraph через Server-Sent Events (SSE). Демонстрирует построение реактивных пайплайнов и работу со статусами выполнения графа.
* **[app/rag.py](app/rag.py)** — Модуль извлечения текстовых данных из документов (PDF, CSV, Excel) и гибридный RAG-поиск. Включает в себя кастомный легковесный алгоритм TF-IDF и косинусного сходства, созданный для минимизации внешних тяжелых зависимостей при локальном запуске.
* **[app/sandbox.py](app/sandbox.py)** — Изолированная среда исполнения Python-кода (Code Interpreter), поддерживающая два режима: безопасное выполнение внутри Docker-контейнера и легковесный режим с предварительной валидацией синтаксического дерева (AST).
* **[tests/test_astream.py](tests/test_astream.py)** — Пример асинхронного интеграционного тестирования графа агентов с использованием обратных вызовов LangChain (`NodeStatusCallbackHandler`) для отслеживания жизненного цикла выполнения узлов.

## 🛠 Ключевой технологический стек системы

* **Язык разработки:** Python 3.11+
* **Базовый фреймворк:** FastAPI (асинхронная обработка запросов, интеграция с ASGI)
* **Агентный фреймворк:** LangChain / LangGraph (построение направленных графов состояний агентов)
* **Интеграция с LLM:** OpenRouter API (асинхронный стриминг ответов моделей DeepSeek, Gemini, etc.)
* **База данных:** SQLite + SQLAlchemy 2.0 (асинхронное взаимодействие через `aiosqlite`)
* **Безопасность песочницы:** Docker SDK + Python AST Scanner

## 🔒 Безопасность и архитектурные решения

1. **Единый стандарт комментариев:** Для удобства ознакомления и детального разбора архитектурных решений весь исходный код демонстрационных модулей снабжен подробными комментариями на **русском языке**.
2. **Изоляция и валидация:** В легковесном режиме выполнения кода используется AST-валидатор, проверяющий синтаксическое дерево на наличие запрещенных импортов и вызовов встроенных функций.
3. **Хранение секретов:** Конфиденциальные данные проекта вынесены в файлы окружения `.env` (пример в `.env.example`) и добавлены в `.gitignore`, исключая риск их утечки в публичный доступ.
